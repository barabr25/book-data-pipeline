## Website
IMDb Top Movies (public, static HTML)

## Tools Used
- Python
- requests
- BeautifulSoup
- pandas

## Data Collected
- Movie title
- Release year
- IMDb rating

## Outcome
Scraped movie data and saved it as a CSV file for analysis.
